Hamming Code SECDED (8,4) Encoder and Decoder

Overview

This project implements an Extended Hamming Code (8,4) SECDED (Single Error Correction, Double Error Detection) system in Verilog HDL. The design generates parity bits to form an 8-bit codeword and uses syndrome generation with overall parity checking to detect and correct transmission errors.

Features

Verilog HDL implementation
Hamming (8,4) SECDED architecture
Single-bit error correction
Double-bit error detection
Syndrome-based error localization
Comprehensive Verilog testbench
Waveform-based functional verification
